## Glossary of Terms and Acronyms

### A
- **AI** - Artificial Intelligence
- **APT** - Advanced Persistent Threat
- **ATT&CK** - Adversarial Tactics, Techniques & Common Knowledge - MITRE Framework

### B
- **BI** - Business Intelligence or in presentation context: Information Security

### C
- **C&C/C2** - Command and Control
- **CCTV** - Closed-Circuit Television
- **CEH** - Certified Ethical Hacker
- **CIA** - Confidentiality, Integrity, Availability
- **CISA** - Certified Information Systems Auditor
- **CISM** - Certified Information Security Manager
- **CISSP** - Certified Information Systems Security Professional
- **CSRF** - Cross-Site Request Forgery
- **CSF** - Cybersecurity Framework
- **CTI** - Cyber Threat Intelligence
- **CVE** - Common Vulnerabilities and Exposures
- **CVSS** - Common Vulnerability Scoring System

### D
- **DAC** - Discretionary Access Control
- **DDoS** - Distributed Denial of Service
- **DFIR** - Digital Forensics and Incident Response
- **DNS** - Domain Name System
- **DoS** - Denial of Service
- **DSR** - Data Subject Request
- **Dumpster Diving** - Dumpster Diving - Searching through trash to find confidential information

### F
- **FIDO2** - Fast IDentity Online 2 - Authentication standard
- **FFuf** - Fuzz Faster U Fool - Fuzzing tool

### G
- **GDPR** - General Data Protection Regulation
- **GIAC** - Global Information Assurance Certification
- **GRU** - Main Intelligence Directorate - Russian military intelligence

### H
- **HID** - Human Interface Device
- **HTTP** - HyperText Transfer Protocol

### I
- **IDS** - Intrusion Detection System
- **IOC** - Indicators of Compromise
- **IP** - Internet Protocol
- **IPS** - Intrusion Prevention System
- **IR** - Incident Response
- **ISO** - International Organization for Standardization
- **ITsec** - IT Security

### K
- **KEV** - Known Exploited Vulnerabilities
- **KYC** - Know Your Customer

### M
- **MAC** - Mandatory Access Control
- **MD5** - Message Digest 5 - Hashing algorithm
- **MFA** - Multi-Factor Authentication
- **MISP** - Malware Information Sharing Platform

### N
- **NFC** - Near Field Communication
- **NIST** - National Institute of Standards and Technology

### O
- **OSCP** - Offensive Security Certified Professional
- **OSINT** - Open Source Intelligence
- **OTX** - Open Threat Exchange

### P
- **PIN** - Personal Identification Number
- **PLA** - People's Liberation Army (China)
- **PUA** - Potentially Unwanted Application

### Q
- **QR** - Quick Response (code)

### R
- **RaaS** - Ransomware-as-a-Service
- **RBAC** - Role-Based Access Control
- **RF** - Radio Frequency
- **R/W** - Read/Write

### S
- **SABSA** - Sherwood Applied Business Security Architecture
- **Sandbox** - Sandbox - Isolated environment for safe testing
- **SHA** - Secure Hash Algorithm
- **SIEM** - Security Information and Event Management
- **SIM** - Subscriber Identity Module
- **SMS** - Short Message Service
- **SOC** - Security Operations Center
- **SOAR** - Security Orchestration, Automation and Response
- **SSL** - Secure Sockets Layer
- **SVR** - Foreign Intelligence Service - Russian civilian intelligence

### T
- **TOGAF** - The Open Group Architecture Framework
- **TOTP** - Time-based One-Time Password
- **TTP** - Tactics, Techniques, Procedures

### U
- **U2F** - Universal 2nd Factor
- **UODO** - Personal Data Protection Office (Poland)
- **UPnP** - Universal Plug and Play
- **URL** - Uniform Resource Locator
- **USB** - Universal Serial Bus

### X
- **XDR** - Extended Detection and Response

### Key Terms and Specialist Vocabulary:

- **Big Three** - CIA Triad (Confidentiality, Integrity, Availability)
- **Crypto-Agility** - Ability to quickly change cryptographic algorithms
- **Credential Stuffing** - Attack using stolen login credentials
- **Cyber Kill Chain** - Cyber kill chain (attack stages model)
- **Dark Web** - Dark web (part of deep web accessible through Tor)
- **Deepfake** - Artificial media created by AI
- **Defense in Depth** - Layered defense
- **Dorks** - Special search engine queries (e.g., Shodan dorks)
- **Face Swap** - Face replacement in video/photo materials
- **Fuzzing** - Security testing technique using random data input
- **Keylogger** - Software recording keystrokes
- **Leetspeak** - Writing style replacing letters with numbers (e.g., 3 instead of E)
- **Low-and-slow** - Long-term, unnoticeable attack technique
- **Malware** - Malicious software
- **Path Traversal** - Directory path attack technique
- **Phishing** - Data theft through impersonation
- **Post-quantum** - Cryptography resistant to quantum computer attacks
- **Prompt Bombing** - Authentication notification bombardment
- **Pyramid of Pain** - Model of difficulty for attackers to change IOC indicators
- **Reverse Proxy** - Proxy server acting on behalf of server
- **Session Cookie** - User session cookie
- **SIM Swapping** - Phone number takeover by changing SIM card
- **Skills Gap** - Shortage of qualified specialists
- **Social Engineering** - Human manipulation
- **Supply Chain** - Supply chain
- **Zero Trust** - "Trust no one" security model
- **Zero-day** - Vulnerability unknown to software vendor
